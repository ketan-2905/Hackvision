Ascendra Design System Specification (v1.0)This document defines the Neo-Brutalist / Industrial Intelligence aesthetic for the Ascendra platform. Use these rules to generate consistent UI for student portals, dashboards, and landing pages.1. Core Visual PrinciplesHigh Contrast: Every element must be clearly separated by heavy borders.Hard Shadows: No soft blurs. Use solid color offsets (4px or 8px) to create depth.Typography as Architecture: Use uppercase, heavy weights (font-black), and extreme letter-spacing (tracking-tighter or tracking-[0.3em]) to create a "technical document" feel.Zero Border Radius: Components should have sharp corners (rounded-none) to maintain the industrial look.2.
